使用 fetch 实现带进度的文件下载
2020-03-18 20:16 label front-end   fetch  

背景故事

在用 Electron 写某个小东西的时候需要实现文件下载功能，因为下载源在国外，国内下载得太慢而被吐槽……于是觉得在下载的时候给一个进度提示会好一些。

fetch() 实现文件下载

const download = (url, save) => {
    fetch(url, {
        method: "GET",
        headers: { "Content-Type": "application/octet-stream" }
    })
        .then(res => res.buffer())
        .then(data => {
            save(data);
        });
};
