为旧款 Synaptics 触摸板添加 Windows Precision 触摸手势

touchpad windows windows precision

Windows Precision 是 Microsoft 在 Windows 10 新增的一系列触摸板手势。笔记本有一块 Windows Precision 触摸板时，可以使用 Windows 自带的三指、四指手势，获得 MacBook 触摸板高达一半的优秀体验。现在市面上的新笔记本，一般都适配了 Windows Precision；而老版的笔记本一般不支持，只能通过触摸板厂商提供的驱动来获得十分有限的体验。然而，如果你正在使用的触摸板是 Synaptics 家的定点装置，那么通过更新驱动就可以获得 Windows Precision 特性。

安装

笔者的笔记本是 n 年前的 HP Probook 4321s(4420s)，搭配的是 Synaptics 的触摸板（并未原生支持 Windows Precision, 在 Windows 10 的系统设置触摸板一项中没有手势设置）。使用 Windows 10 自带的 Synaptics 定点装置驱动时，在 UWP 应用下无法使用任何触摸板手势，于是乎打算更新一下触摸板的驱动。偶然间心血来潮，想找找是否有能让触摸板支持 Windows 10 手势或者模拟手势的驱动，于是就找到了一个支持 Windows Precision 的 Syanptics 驱动。安装之后效果如下：

试用了一下简直是爽的不行。为此特别开一篇文记录一下过程。

以下是 Synaptics 触摸板安装支持 Windows Precision 驱动的步骤：

  1. 下载来自美帝良心想的适用于 Lenovo Thinkpad Synaptics 定点装置的驱动：https://download.lenovo.com/pccbbs/mobiles/n1mgx14w.zip

  2. 解压到某个文件夹，打开设备管理器，在“鼠标和其它指针设备”中找到带有 Synaptics 字样的触摸板设备；

  3. 点击右键 ==> 更新驱动程序，依次选择“浏览我的计算机以查找驱动程序文件” ==> “让我从计算机上的可用驱动程序列表中选取” ==> “从磁盘安装”

  4. 定位到下载的文件中的 autorun.inf

  5. 选择 “Synaptics Pointing Device” (注意不要选择 ThinkPad 的那一个)，下一步安装

  6. 重启电脑

重启后打开系统设置的触摸板设置中，如果看到了“您的电脑有一块精确式触摸板”字样，则说明安装成功。

后记

乡下 Windows 用户终于体验到了三指手势的好用之处……现在在 UWP 应用里也可以使用触摸板手势了。

这个驱动虽然来自联想，但是在我的 HP 上也能用，而且只要不是特别老的触摸板，兼容性一般都不错。不过因为我的触摸板最多似乎只支持 3 点触控，所以就只能使用三指手势，并且灵敏度也不是那么高。不过既然已经是很老的机子了，所以也就不要什么自行车了。
